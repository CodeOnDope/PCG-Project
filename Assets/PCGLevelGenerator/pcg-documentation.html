PCG Level Generator Documentation

Welcome to the PCG Level Generator documentation. This powerful Unity asset provides an intuitive system for procedural level generation in 2D games.

With three distinct generation approaches, flexible customization options, and a visual editor, PCG Level Generator is designed to save developers time while creating engaging and dynamic game worlds.

Key Features:
  • Three generation modes: Fully Procedural, Hybrid Procedural, and User-Defined Layout
  • Visual Level Designer for custom layouts
  • BSP (Binary Space Partitioning) algorithm for natural-looking spaces
  • Room templates support for consistent designs
  • Automatic entity placement (players, enemies, decorations)
  • Full Unity Tilemap integration
  • One-click setup (ready in 10 seconds)

Installation

There are two ways to install PCG Level Generator in your Unity project:

Method 1: Unity Asset Store

  1. Purchase PCG Level Generator from the Unity Asset Store
  2. Open your Unity project and go to Window > Package Manager
  3. Select "My Assets" from the Packages dropdown
  4. Find PCG Level Generator and click "Download" then "Import"
  5. In the Import window, ensure all components are selected and click "Import"

Method 2: Manual Import

  1. Download the PCG Level Generator .unitypackage file from your Asset Store downloads
  2. Open your Unity project
  3. Go to Assets > Import Package > Custom Package...
  4. Locate and select the downloaded .unitypackage file
  5. Ensure all components are selected in the Import window and click "Import"
Note: After importing, you'll find the PCG Level Generator folder in your project's Assets directory. The package is fully self-contained and doesn't add any code outside its folder.

Quick Start

Get up and running with PCG Level Generator in just a few steps:

One-Click Setup

  1. Open your Unity scene
  2. Go to Tools > PCG Level Generator > One-Click Setup
  3. The system will create all required components in your scene
  4. Select the newly created HybridLevelGenerator GameObject
  5. In the Inspector, choose a generation mode
  6. Click "Generate Level" to create your first procedural level
Pro Tip: For quick iteration, you can adjust settings in the Inspector and regenerate the level at any time by clicking "Generate Level" again.

Working with the Visual Designer

If you prefer to design your level layout manually:

  1. Go to Tools > PCG Level Generator > Welcome Screen
  2. Click "Open Visual Level Designer"
  3. Use the designer to create and connect rooms
  4. Click "Create Scene Objects & Generate" to implement your design
Important: Make sure your scene has a Grid GameObject with appropriate Tilemap components before generating levels. The one-click setup takes care of this automatically.

System Requirements

PCG Level Generator is designed to be lightweight and compatible with a wide range of Unity versions and platforms.

Unity Version

  • Unity 2020.3 LTS or newer
  • Fully tested on Unity 2020.3 LTS, 2021.3 LTS, and 2022.3 LTS

Required Packages

  • 2D Tilemap Editor (included in Unity's 2D packages)
  • 2D Tilemap Extras (recommended but optional)

Supported Platforms

PCG Level Generator works on all platforms that Unity supports. It has been tested on:

  • Windows, Mac, and Linux (for development)
  • iOS and Android
  • WebGL
  • Consoles (Switch, PlayStation, Xbox)

Performance Considerations

The generator is optimized for runtime and editor use. Generation time depends on level size and complexity:

  • Small levels (50x50): ~0.1 seconds
  • Medium levels (100x100): ~0.3 seconds
  • Large levels (200x200): ~1 second
Note: Generation is typically performed at level load time or during scene transitions, so it doesn't affect gameplay performance.

Fully Procedural Mode

The Fully Procedural mode uses Binary Space Partitioning (BSP) to generate levels automatically with rectangular rooms and corridors.

How It Works

  1. The generator divides the available space using BSP
  2. Rectangular rooms are placed within each partition
  3. Corridors are generated to connect rooms using a minimum spanning tree algorithm
  4. Tiles are applied to the grid based on the generated layout
  5. Entities (player, enemies, decorations) are placed in appropriate locations

Key Settings

  • Level Width/Height: The maximum dimensions of the generated level
  • Min Room Size: Minimum width/height for generated rooms
  • Max Iterations: Number of BSP splits to perform
  • Room Padding: Minimum space between rooms
  • Corridor Width: Width of corridors connecting rooms
Best For: Classic roguelike dungeons, caves, office buildings, or any environment with regular rectangular rooms.

Hybrid Procedural Mode

Hybrid Procedural mode combines BSP splitting with varied room types including rectangles, L-shapes, and custom room templates for more diverse and interesting levels.

How It Works

  1. The generator divides the available space using BSP
  2. For each partition, it randomly chooses a room type (rectangle, L-shape, or template)
  3. Rooms are placed according to their type-specific rules
  4. Corridors are generated to connect all rooms
  5. Tiles and entities are placed based on the generated layout

Key Settings

  • L-Shape Probability: Chance (0-1) that a room will be L-shaped
  • Min/Max L Leg Ratio: Controls the proportions of L-shaped rooms
  • Room Template Probability: Chance (0-1) that a room will use a template
  • Room Template Prefabs: List of prefabs to use as room templates
Best For: Games that need varied architecture like RPGs, adventure games, or strategy games with diverse environments.

User-Defined Layout Mode

User-Defined Layout mode gives you complete control over level design through the Visual Level Designer, while still handling the technical aspects of level creation automatically.

How It Works

  1. Design your level layout using the Visual Level Designer
  2. Place and connect rooms of different types (rectangle, L-shape, template)
  3. Set room properties (size, templates, connections)
  4. Create Scene Objects to generate RoomNode components
  5. Generate the level based on your design

Key Features

  • Visual Editor: Intuitive drag-and-drop interface
  • Room Types: Choose from rectangle, L-shape, or template rooms
  • Connections: Define corridors between rooms
  • Properties: Set size and appearance for each room
  • Scene Integration: Converts visual design to Unity scene objects
Best For: Hand-crafted level designs, puzzle games, tutorial levels, or any scenario where you need precise control over the layout.
